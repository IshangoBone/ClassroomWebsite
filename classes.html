Classes We Teach

Click on each course to learn more about what students will study, practice, and build throughout the year.

Introduction to Engineering Design

Students explore the engineering design process by identifying problems, brainstorming solutions, building prototypes, and improving their ideas through testing.

  • Learn the steps of the design process
  • Create and test prototype solutions
  • Develop teamwork and problem-solving skills
Introduction to Computer Science

This course introduces students to the fundamentals of computer science through programming, logic, and computational thinking.

  • Practice writing and debugging code
  • Build problem-solving and logical thinking skills
  • Explore how computer science connects to the real world
Computing Foundations for a Digital Age

Students examine how computing impacts communication, work, creativity, and daily life while building digital literacy and technical confidence.

  • Explore digital systems and online tools
  • Understand how technology affects society
  • Build confidence using modern computing tools
AP Computer Science A

This advanced course focuses on programming in Java and prepares students for the AP Computer Science A exam through practice, projects, and deeper analysis.

  • Write Java programs using core programming concepts
  • Strengthen algorithmic thinking and debugging skills
  • Prepare for the AP exam with rigorous coding practice
Personal Finance

Students build practical money-management skills that will help them make informed financial decisions in school, college, and adult life.

  • Learn budgeting, saving, and spending strategies
  • Understand credit, loans, and financial responsibility
  • Explore investing and long-term planning basics
